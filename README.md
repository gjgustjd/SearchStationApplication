# SearchStationApplication
## 🤗소개
외부 API를 통해 지하철 역 정보를 요청하여 
선택한 지하철 역을 리스트로 보여주는 간단한 어플입니다.

## 🔧적용 기술 
- Retrofit2
- AAC를 활용한 MVVM 패턴
- StateFlow+DataBinding을 이용한 UI
- Coroutine을 이용한 비동기 처리
- Hilt를 사용한 의존성 주입
- Room을 사용하여 내부 데이터 저장
